# 门外审堂学习系统 — 真机测试计划

## 1. 测试概述

| 项目 | 说明 |
|------|------|
| 测试目标 | 验证门外审堂学习系统 MVP 核心功能在真机环境下的正确性、稳定性和用户体验 |
| 测试版本 | V1.0.0 (Build 20260617) |
| 技术栈 | Tauri 2.0 + React 18 + Rust + SQLite |
| 测试环境 | 桌面端 (Linux/macOS/Windows) + 移动端 (Android/iOS) |
| 测试周期 | 3-5 个工作日 |

## 2. 构建与安装

### 2.1 桌面端构建

```bash
# Linux
npm run tauri build
# 产物: src-tauri/target/release/bundle/appimage/*.AppImage
#        src-tauri/target/release/bundle/deb/*.deb

# macOS
npm run tauri build
# 产物: src-tauri/target/release/bundle/dmg/*.dmg

# Windows
npm run tauri build
# 产物: src-tauri/target/release/bundle/msi/*.msi
```

### 2.2 移动端构建（需交叉编译环境）

```bash
# Android
npm run tauri android build
# 产物: src-tauri/gen/android/app/build/outputs/apk/

# iOS (需 macOS + Xcode)
npm run tauri ios build
# 产物: src-tauri/gen/apple/target/release/bundle/ios/*.ipa
```

### 2.3 安装验证

| 验证项 | 预期结果 |
|--------|----------|
| 安装包大小 | < 30MB |
| 安装过程 | 无报错，桌面快捷方式/开始菜单正常创建 |
| 首次启动 | 显示加载动画 → 进入引导页 |
| 应用图标 | 正确显示（非空白/默认图标） |
